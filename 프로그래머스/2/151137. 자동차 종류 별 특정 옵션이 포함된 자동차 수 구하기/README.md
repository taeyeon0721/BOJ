### 문제 풀이 
- 처음에 내가 틀렸던 이유: '%시트'라고 작성함
- 이렇게 하면 안되는 이유? 한 행을 하나의 문자열로 보면<br> '시트'라는 단어가 포함되어 있어도 OO시트로 끝나지 않는게 있음!<br>
  ex) 가죽시트,열선시트,후방카메라
- '시트'라는 단어가 하나 이상 들어간 행을 구하는 것이기 때문에 '%시트%' 이렇게 작성해야함
